# Craftable Spawner and Eggs

This mod does exactly as the name suggests and brings crafting recipes for the spawner and spawn eggs with it's own difficulty, making each recipe fair and fits the mob.  

Every single spawn egg for each version has been added with its own crafting recipe.  

For mod issues, please see the **[GitHub Issues](https://github.com/its-Jaxx/Craftable-Spawner-and-Eggs/issues)** page
